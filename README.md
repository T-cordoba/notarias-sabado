# Notarías Sábado 🇨🇴

Aplicación web que facilita la consulta de notarías con atención sabatina en Colombia, usando directamente la información oficial del [Supernotariado](https://www.supernotariado.gov.co/notarias-sabado/).

## ¿Qué hace?

El Supernotariado publica semanalmente qué notarías atienden los sábados, pero su sitio oficial es difícil de navegar y la información aparece duplicada y desactualizada. Esta app resuelve eso:

- Consulta los datos oficiales en tiempo real desde el Supernotariado
- Filtra registros duplicados y entradas sin información útil
- Permite buscar por **fecha**, **departamento** y **ciudad**
- Muestra dirección, teléfono, correo y horario de cada notaría
- Genera un enlace directo a Google Maps a partir de la dirección registrada

## Uso

Seleccioná la fecha (solo sábados disponibles), el departamento y la ciudad, y la app muestra automáticamente las notarías habilitadas para esa combinación.

## Datos

Toda la información proviene directamente del Supernotariado — esta app no almacena ni modifica los datos, solo los organiza y presenta de forma más accesible.

Las direcciones y enlaces a Maps pueden no ser exactos ya que los datos del Supernotariado a veces contienen errores de digitación. Se recomienda verificar la dirección antes de ir.
